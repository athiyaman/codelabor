
 
 
 
[진행 현황 보고, 설명회 진행할 것]
업체로 하여금 진행상황을 수행사 PMO, 고객사 PMO, 개발 리더 입회하에 보고케할 것 (참석자가 누구인지 정확히 전달할 것)
 TA, SA, 개발 리더는 진행 현황 보고 내용 중, 기술 부문에 대한 크로스 체크를 진행할 것
진행현황 보고 시 아래 내용 포함 시킬 것
- 전체 목표 대비 진행 현황 (Feature 별로 완료한 것, 진행할 것)
- 기술 플랫폼 변경으로 인한 아키텍처 변경 내용 (아키텍처 정의서)
 
[점검에 필요한 체크리스는 설명회 후, 작성할 것]
설명회 요청 시, PM님이 확인하고 싶은 것을 간략히 기재하여 준비케 할 것
설명회 요청 시, SA가 확인하고 싶은 것을 간략히 기재하여 준비케 할 것
설명회 진행 중에 추가 확인 사항을 메모할 것
설명회 종료 후, 메모한 내용으로 질의서를 작성, 메일로 전송하고 회신케 할 것
 
[향후 정기 점검 방법]
기 작성된 질의서 중, 정기적으로 확인이 필요한 내용을 정리, 정기 현황 보고 시 요청할 것
 
[기타 Action Item]
설명회 후, 아래와 같은 내용을 수행토록 요청할 것 
- 품질 기준에 대한 명확한 기준 제시: PMD Ruleset 제공하여 준수케 할 것
- 보안 기준에 대한 명확한 기준 제시: 보안 체크리스트의 내용 발췌, 전달하여 준수케 할 것 (원본은 대외비이므로 제공 불가)
